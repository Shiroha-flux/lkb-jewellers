# Rose — Hover Images to Generate

**Ring type:** Fluted cigar band / croissant ring, pillowy wide band, no center stone  
**Target files:** `yellow_2.jpg`, `white_2.jpg`, `rose_2.jpg`

## OpenArt Workflow (per color)

1. Open OpenArt → Edit Image
2. Upload: `yellow_1.jpg` / `white_1.jpg` / `rose_1.jpg` (the ring render)
3. Prompt:
   > Place this ring inside an open LKB jewellery box. The box is matte black velvet exterior, cream/beige suede interior, white "KB" logo on the back panel. Ring resting flat or slightly angled on cream pillow insert. Grey studio background, frontal eye-level shot, soft natural lighting, photorealistic.
4. Save output as: `yellow_2.jpg` / `white_2.jpg` / `rose_2.jpg`

## Note
Rose adalah wedding band (cigar/croissant style) — tidak ada center stone.
Ring bisa diletakkan flat atau sedikit miring di atas pillow, bukan berdiri tegak.
